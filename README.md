# 2D Collision Simulator

Physics-based 2D particle simulation written in C++ using GLFW.

## Features
- Elastic collisions
- Boundary interactions
- Real-time rendering
- G-force directed to the bottom

## Build

g++ main.cpp -lglfw -framework OpenGL -o simulator

## How to replicate

Install the GLFW library and add to the path of your compiler, then compile and run the code.
